Crypteco

A collaboration platform for giving people insights on business ecosystems in an easy-to-update, content-rich and interactive way.

More insights

Want to know more about the Chinese bike sharing market?

If you have some knowledge about the market, start your ecosystem by dragging organisations from the libraries to the board. In case you have no knowledge about the market, but would like to learn more, request a researcher to create an ecosystem for you.

Create relationships

Want to have some context?

Show the relationships between one or more stakeholders. Connect them via a 1-to-1 junction or by grouping them.

Smart data

Need more data?

Click on the blue info symbol and you will see, there is more data available, in case you are missing data, feel free to add it. More smart data, that can be used to visualize relative sizes of organisations.

Collaborate together

Excited to share your work?

Crypteco allows to work in private mode on an ecosystem, but you can also work with several people up to the whole public on an ecosystem. Share your ecosystem with whoever you want, by posting the link to the board on social media or inviting people via email link to see it.
